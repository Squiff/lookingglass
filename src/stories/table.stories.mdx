## Usage

`<Table>` is a styled wrapper around the standard `<table>` element.
It can be used as a direct replacement for `<table>`.

## Header Color

Table headers have no background by default.
Use backgroundColor, color and borderColor utilities to style the header row.

Examples in this section will use the `primary` color.

```jsx
<thead>
    <Lookingglass backgroundColor="primary" color="white" borderColor="primary">
        <tr>
            <th>Headers 1</th>
            <th>Headers 2</th>
            <th>Headers 3</th>
            <th>Headers 4</th>
        </tr>
    </Lookingglass>
</thead>
```
