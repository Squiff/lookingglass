## Opening and Closing

You can add a close button by setting `closeBtn = true`.
When the close button is clicked the `Alert` will transition out and unmount.

<Canvas>
    <Story story={stories.CloseButton} />
</Canvas>

### Controlled

By passing in the `show` prop, `<Alert>` becomes a **_controlled_** component. This passes responsibility for showing/hiding the alert to the user.

Set `show` to true/false to hide/show the alert.

When using `closeBtn` with a controlled `Alert`, it will trigger the `onClose` callback, which should be used to
set your state.
