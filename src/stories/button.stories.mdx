## Colors

Apply color by setting the `color` prop.
A button will default to solid color with the appropriate dark/white text. A darkening effect will be applied on hover.

## No Style

Set `btnStyle="none"` for an 'unstyled' button.
Base `Button` styles such as font-size and pointer cursor are still retained (so technically it is still styled!).

<Canvas>
    <Story story={stories.NoStyle} />

#### Adding Hover Effect

When `bynStyle="none"` a `hoverEffect` can be added. Three effects are provided:

-   **dark**: Translucent dark background applied
-   **light**: Translucent white background applied
-   **opacity**: Opacity added

Setting the standard `disabled` attribute will add `opacity` to the button and remove any hover styles.

    <Story story={stories.Disabled} />

## Links

`Button` will render as an `<a>` element when a `href` provided.

The `disabled` flag on a link button will:

-   Mark element as `aria-disabled`
-   add `tabindex="-1"` to prevent tabbing
-   add `preventDefault` to the click event to stop navigation to the URL
