## Basics

A Switch shows components based on the status passed to it.

#### Using Values

Set a `value` on the `Switch` and any component in the corresponding `Switch.Case` that has a matching value (using `===`) will be rendered.

```jsx
<Switch value="success">
    <Switch.Case value="success">
        <Alert color="success">Complete!</Alert>
    </Switch.Case>
    <Switch.Case value="error">
        <Alert color="error">Error!</Alert>
    </Switch.Case>
    <Switch.Case value="loading">
        <Spinner />
    </Switch.Case>
</Switch>
```

#### Using booleans

The `value` for `Switch.Case` can also accept booleans. If set to true then the component will be rendered.
