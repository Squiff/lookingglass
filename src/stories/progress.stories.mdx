## Setting Progress

Set the percentage completion using the `progress` prop. It expects a value between 0 and 100.
When the progress is updated it will transition to the new value.

    <Story story={stories.setProgress} />

## Indeterminate Progress

You can set the `indeterminate` flag to enable an animation on the progress bar.
This is useful when the current progress of an operation is unknown.
