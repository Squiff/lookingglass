A `List` is an unordered list with opinionated styles. A `List` is made up of the following subcomponents:

-   `List.Item`
-   `List.Button`
-   `List.Link`

## Links and buttons

`List.Button` and `List.Link` forward all props to `<button>` and `<a>` elements respectively.

The benefit of using `List.Button` and `List.Link` is that they will have
consistant styling with a `List.Item`. They will also have a hover effect added.
This is particularly useful when used as a navigation tool.

Lists like this can be combined with `<Drawers>` to create a a sliding side menu or in `<NavBar>` dropdowns.

#### Divider

Adding a divider is as simple as adding an `<hr/>` in `List.Item`

```jsx
<List.Item>
    <hr />
</List.Item>
```

## Item Composition

Use composition to alter `List.Item` styles.

```jsx
const MyListItem = ({ children, ...props }) => (
    <Lookingglass paddingX="1" paddingY="3" fontSize="1.25">
        <List.Item {...props}>{children}</List.Item>
    </Lookingglass>
);
```
