## Pre-Defined Positions

Using the `absolute` prop you can position an element in one of nine positions, without the need for additional transforms.

Values are: `top-left`,
`top-middle`,
`top-right`,
`middle-left`,
`middle`,
`middle-right`,
`bottom-left`,
`bottom-middle` and `bottom-right`
