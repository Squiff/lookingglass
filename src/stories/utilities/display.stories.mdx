## Display Values

The `display` prop accepts string values of `block`, `inline-block` and `none`.

### Responsive

Display can also be passed an object of shape `{[breakpoint]:[display value]}`.
This will set the display value from that breakpoint **upwards**.

Here we have two elements with the following display values

**`{s: 'none', l:'block'}`** This will not display at the small breakpoint, but will display from the large breakpoint.

**`{s: 'block', l:'none'}`** This will display at the small breakpoint, but will not display from the large breakpoint.
