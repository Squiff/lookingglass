## Border Width

`border`, `borderTop`, `borderRight`, `borderBottom`, `borderLeft` accept values `0` through `4`
, which equates to the pixel width of the related border.

`border` is applied to all sides unless overridden by one of the directional border props. In this
example we set `border="1"` and `borderLeft="4"`

## Border Radius

The `borderRadius` prop accepts values '0' through '3' to adjust the border radius.
A value of `rounded` can also be applied to completely round the edges.
