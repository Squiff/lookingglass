## Support

Scrollbar utilities use the following CSS Properties for styling

-   **:-webkit-\***
-   **scrollbar-color**
-   **scrollbar-width**

As such, support is only available for Webkit browsers and browsers that support standard CSS styling
(Currently only firefox - see [CanIUse](https://caniuse.com/?search=scrollbar-color))

## Scrollbar Color

A Dark and Light variant can be selected via the `scrollbarColor` prop.

    <Story story={stories.Colors} />

## Scrollbar Size

Set `scrollbarSize="s"` for a small scrollbar.
`scrollbarSize="none"` will remove the scrollbar altogether (but the container will remain scrollable)

    <Story story={stories.Sizes} />

## Rounded

Setting `scrollbarStyle="rounded` will round the scrollbar thumb
