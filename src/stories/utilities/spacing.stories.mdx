# Using Spacing

Margin and padding can be adjusted using values `0` to `5`, which range from `0` to `32px`.
Margin props also accept a value of `auto`.

If more than one prop is passed that affects a particular direction, then they will be prioritized, from lowest to highest:

margin/padding -> Direction X/Y -> Direction Left/Top/Right/Bottom
