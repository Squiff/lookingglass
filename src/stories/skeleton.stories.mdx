## Usage

A skeleton is a placeholder that can be used when you know the shape of the content that will be loaded.
Use the `className` or `style` properties to set the shape of your expected content by applying height, margins, border radius etc.

Setting the `dark` prop changes the color scheme of the skeleton to be more suitable for darker backgrounds.
