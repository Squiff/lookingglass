## Custom Styles
Custom styles can be applied to spinners by setting className or the relevant style.

- **Color**:  Spinners use `currentcolor`, so you can assign any class/style that sets color
- **Size**: assign a class/style that sets the `height` and `width`

Note that when setting size, you should still use the size prop to get the most appropiate stroke width to size ratio i.e. larger spinners should use `size="l"`
